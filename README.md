## Image files

All images are inside:

```text
assets/images/
```

Current files:

```text
hero-photo.jpg      # Main hero image
cover-photo.jpg     # Tap-to-reveal image
walking-photo.jpg   # Gallery image
og-preview.jpg      # Social media/link preview image
```

To replace a photo later, add a new image to `assets/images/` and update the matching filename in `index.html`.
