# Kefin & Ludhiya Wedding Website

Static GitHub Pages wedding invitation with soft animations inspired by the reference site.

## What is included

- Animated full-screen hero image
- Tap-to-reveal save-the-date card
- Soft chime on reveal / music button
- Confetti burst
- Animated countdown numbers
- Ceremony and reception cards with Google Maps links
- Family details
- Gallery section
- Open Graph preview image for WhatsApp/iMessage/Facebook link previews
- No RSVP section

## Image files

All images are inside:

```text
assets/images/
```

Current files:

```text
hero-photo.jpg      # Main hero image
cover-photo.jpg     # Tap-to-reveal image
walking-photo.jpg   # Gallery image
og-preview.jpg      # Social media/link preview image
```

To replace a photo later, add a new image to `assets/images/` and update the matching filename in `index.html`.

## GitHub Pages upload steps

1. Unzip this folder.
2. Upload the contents directly into the root of your `invite` repository.
3. Make sure `index.html`, `styles.css`, `script.js`, and `assets/` are at the top/root level of the repository.
4. Go to **Settings → Pages**.
5. Use **Deploy from a branch** → `main` → `/root`.
6. Wait 1–3 minutes, then refresh your site.

Your expected URL is:

```text
https://kefsaj.github.io/invite/
```
