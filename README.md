# Wedding site

A static wedding invitation site designed for GitHub Pages.

## Files

- `index.html` — page structure and invitation text
- `styles.css` — colors, layout, floral styling, responsive design
- `script.js` — countdown timer, mobile menu, reveal animation
- `assets/images/` — add photos here later

## How to customize

1. Open `index.html`.
2. Replace the RSVP email/link in the RSVP section.
3. Add photos to `assets/images/`.
4. Replace the gallery placeholders with image tags, for example:

```html
<img src="assets/images/photo1.jpg" alt="Kefin and Ludhiya" />
```
