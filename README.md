# Kefin & Ludhiya Wedding Website

This version uses a **luxury / modern / minimal** style with one featured image, refined typography, generous spacing, and a cleaner page structure.

## Included sections

- Main photo hero section
- Short welcome / couple note
- Mark 10:9 verse
- Countdown
- Ceremony details
- Reception details
- Google Maps links
- Wedding day schedule
- Expanded family details

## Where to change the main photo

The main image is here:

```text
assets/images/hero-photo.jpg
```

To replace it later, add your new image to that same folder and name it:

```text
hero-photo.jpg
```

No code changes are needed if the filename stays the same.

## Deploy on GitHub Pages

1. Create a GitHub repository, such as `wedding-invite`.
2. Upload all files and folders from this package.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save. GitHub will give you a website link.
